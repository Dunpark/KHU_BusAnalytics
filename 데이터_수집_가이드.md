## 1. 서울시 우리마을가게 상권분석서비스 데이터

### 📍 다운로드 사이트
- **사이트명:** 서울 열린데이터광장
- **URL:** https://data.seoul.go.kr
- **검색 키워드:** "우리마을가게 상권분석서비스" 또는 "골목상권분석정보"

### 📥 다운로드 방법
1. 서울 열린데이터광장 접속 (회원가입/로그인 필요)
2. 상단 메뉴: **공공데이터 > 인기그룹데이터 > 골목상권분석정보**
3. 또는 검색창에 "우리마을가게 상권분석서비스" 검색
4. 각 카테고리별로 분기별 데이터 다운로드

### 📊 다운로드할 데이터 카테고리 (6개)

#### 1.1 상권-점포 데이터
- **데이터셋명:** 서울시 우리마을가게 상권분석서비스(상권-점포)
- **기간:** 2019년 1분기 ~ 2024년 4분기 (각 분기별 파일)
- **주요 변수:** 상권 코드, 상권 명칭, 점포 수, 업종 분류
- **다운로드 형식:** CSV 또는 Excel 선택
- **파일명 예시:** `상권-점포_2019Q1.csv`, `상권-점포_2019Q2.csv` ... `상권-점포_2024Q4.csv`
- **총 파일 수:** 24개 (6년 × 4분기)

#### 1.2 상권-임대료 데이터
- **데이터셋명:** 서울시 우리마을가게 상권분석서비스(상권-임대료)
- **기간:** 2019년 1분기 ~ 2024년 4분기
- **주요 변수:** 상권 코드, 상권 명칭, 평균 임대료
- **다운로드 형식:** CSV 또는 Excel
- **파일명 예시:** `상권-임대료_2019Q1.csv` ... `상권-임대료_2024Q4.csv`
- **총 파일 수:** 24개

#### 1.3 상권-공실률 데이터
- **데이터셋명:** 서울시 우리마을가게 상권분석서비스(상권-공실률)
- **기간:** 2019년 1분기 ~ 2024년 4분기
- **주요 변수:** 상권 코드, 상권 명칭, 공실률 (%)
- **다운로드 형식:** CSV 또는 Excel
- **파일명 예시:** `상권-공실률_2019Q1.csv` ... `상권-공실률_2024Q4.csv`
- **총 파일 수:** 24개

#### 1.4 상권-매출 데이터
- **데이터셋명:** 서울시 우리마을가게 상권분석서비스(상권-매출)
- **기간:** 2019년 1분기 ~ 2024년 4분기
- **주요 변수:** 상권 코드, 상권 명칭, 업종별 매출
- **다운로드 형식:** CSV 또는 Excel
- **파일명 예시:** `상권-매출_2019Q1.csv` ... `상권-매출_2024Q4.csv`
- **총 파일 수:** 24개

#### 1.5 상권-인구 데이터
- **데이터셋명:** 서울시 우리마을가게 상권분석서비스(상권-인구)
- **기간:** 2019년 1분기 ~ 2024년 4분기
- **주요 변수:** 상권 코드, 상권 명칭, 연령대별 유동인구, 성별 유동인구, 생활인구
- **다운로드 형식:** CSV 또는 Excel
- **파일명 예시:** `상권-인구_2019Q1.csv` ... `상권-인구_2024Q4.csv`
- **총 파일 수:** 24개

#### 1.6 상권-소득 데이터
- **데이터셋명:** 서울시 우리마을가게 상권분석서비스(상권-소득)
- **기간:** 2019년 1분기 ~ 2024년 4분기
- **주요 변수:** 상권 코드, 상권 명칭, 평균 소득
- **다운로드 형식:** CSV 또는 Excel
- **파일명 예시:** `상권-소득_2019Q1.csv` ... `상권-소득_2024Q4.csv`
- **총 파일 수:** 24개

### ⚠️ 주의사항
- 데이터 업데이트 주기: 매 분기 2개월 후 업데이트 (예: 1분기 데이터는 5월 말 업데이트)
- 상권 코드가 모든 데이터셋에서 공통 키로 사용되므로 반드시 확인
- 데이터 형식은 CSV 권장 (분석 용이성)

## 2. 서울시 골목상권분석정보 데이터

### 📍 다운로드 사이트
- **사이트명:** 서울 열린데이터광장
- **URL:** https://data.seoul.go.kr
- **검색 키워드:** "골목상권분석정보"

### 📥 다운로드 방법
1. 서울 열린데이터광장 접속
2. 검색창에 "골목상권분석정보" 검색
3. 분기별 데이터 다운로드

### 📊 데이터 내용
- **기간:** 2019년 이후 분기별 (확인 필요)
- **주요 변수:**
  - 점포수 (사업자등록번호 기반)
  - 추정매출 (신한카드, 비씨카드 데이터 기반)
  - 유동인구 (서울시, KT 데이터 기반)
  - 임대시세 (서울신용보증재단 데이터 기반)
- **다운로드 형식:** EXCEL, CSV, OPEN API, SHP(공간영역)
- **용도:** 우리마을가게 데이터와 교차 검증, 개별 매장 추적 가능성

## 3. 한국부동산원 상업용 부동산 임대동향조사 데이터

### 📍 다운로드 사이트
- **사이트명:** 공공데이터포털
- **URL:** https://www.data.go.kr
- **검색 키워드:** "상업용부동산 임대동향조사"

### 📥 다운로드 방법
1. 공공데이터포털 접속 (회원가입/로그인 필요)
2. 검색창에 "상업용부동산 임대동향조사" 검색
3. 중대형상가 공실률, 소규모상가 공실률 각각 다운로드

### 📊 다운로드할 데이터

#### 3.1 중대형상가 공실률 데이터
- **데이터셋명:** 상업용부동산 임대동향조사 - 중대형상가 공실률
- **기간:** 2019Q1 ~ 2024Q4 (확인 필요)
- **지역:** 서울시 지역별
- **주요 변수:** 지역별 공실률, 분기별 데이터
- **데이터셋 번호:** 확인 필요 (공공데이터포털에서 검색 시 확인)
- **URL 예시:** https://www.data.go.kr/data/15069735/fileData.do (확인 필요)

#### 3.2 소규모상가 공실률 데이터
- **데이터셋명:** 상업용부동산 임대동향조사 - 소규모상가 공실률
- **기간:** 2019Q1 ~ 2024Q4 (확인 필요)
- **지역:** 서울시 지역별
- **주요 변수:** 지역별 공실률, 분기별 데이터
- **데이터셋 번호:** 확인 필요
- **URL 예시:** https://www.data.go.kr/data/15069726/fileData.do (확인 필요)

### ⚠️ 주의사항
- 중대형/소규모 상가의 정의 확인 필요
- 지역 구분 단위 확인 (구 단위, 상권 단위 등)
- 상권 코드와의 매칭 방법 확인

## 4. 서울시 행정동별 인구 및 소득 데이터

### 📍 다운로드 사이트
- **사이트명:** 통계청 KOSIS (국가통계포털)
- **URL:** https://kosis.kr
- **검색 키워드:** "서울시 행정동별 인구", "서울시 행정동별 소득"

### 📥 다운로드 방법
1. KOSIS 접속
2. 검색창에 "서울시 행정동별 인구" 또는 "서울시 행정동별 소득" 검색
3. 연도별 또는 분기별 데이터 다운로드

### 📊 데이터 내용
- **기간:** 2019년 ~ 2024년 (연간 또는 분기별)
- **지역:** 서울시 행정동별
- **주요 변수:**
  - 행정동별 주거인구
  - 행정동별 평균 소득
  - 행정동별 소비 구조
- **다운로드 형식:** Excel, CSV

### ⚠️ 주의사항
- 상권 데이터와 행정동 데이터의 매칭 필요
- 행정동 코드 또는 명칭으로 매칭
- 데이터 업데이트 주기 확인
