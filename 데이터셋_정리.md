# 프로젝트 데이터셋 정리

**프로젝트 주제:** 젠트리피케이션 & 폐업 위험도 조기 경보 시스템  
**분석 기간:** 2019Q1 ~ 2024Q4  
**분석 지역:** 서울시  
**작성일:** 2025년

---

## 1. 서울시 우리마을가게 상권분석서비스 데이터

### 1.1 상권-점포 데이터
- **출처:** 서울 열린데이터광장 (data.seoul.go.kr)
- **데이터셋명:** 서울시 우리마을가게 상권분석서비스(상권-점포)
- **기간:** 2019년 1분기 ~ 2024년 4분기 (분기별 데이터)
- **지역:** 서울시 전체 상권
- **주요 변수:**
  - 상권 코드 및 명칭
  - 점포 수
  - 업종 분류
  - 분기별 데이터
- **프로젝트 활용 계획:**
  - 매장별 폐업 여부 판단을 위한 기준 데이터
  - 업종별, 상권별 점포 수 변화 추이 분석
  - 폐업 위험도 모델링의 종속변수 생성 (점포 수 감소 = 폐업 지표)
- **주의사항:**
  - 점포 수 데이터만으로는 개별 매장의 폐업을 직접 확인하기 어려울 수 있음
  - 업종 분류가 세분화되어 있는지 확인 필요
  - 데이터 업데이트 주기 확인 (분기별 업데이트 시점)

### 1.2 상권-임대료 데이터
- **출처:** 서울 열린데이터광장 (data.seoul.go.kr)
- **데이터셋명:** 서울시 우리마을가게 상권분석서비스(상권-임대료)
- **기간:** 2019년 1분기 ~ 2024년 4분기 (분기별 데이터)
- **지역:** 서울시 전체 상권
- **주요 변수:**
  - 상권 코드 및 명칭
  - 평균 임대료
  - 분기별 데이터
- **프로젝트 활용 계획:**
  - 젠트리피케이션 분석의 핵심 지표

  - 코로나19 전후 임대료 변동 패턴 분석 필요

## 2. 서울시 골목상권분석정보 데이터

- **출처:** 서울 열린데이터광장 (data.seoul.go.kr)
- **데이터셋명:** 골목상권분석정보
- **기간:** 2019년 이후 분기별 데이터 (확인 필요)
- **지역:** 서울시 전체 상권
- **주요 변수:**
  - 점포수 (사업자등록번호 기반)
  - 추정매출 (신한카드, 비씨카드 데이터 기반)
  - 유동인구 (서울시, KT 데이터 기반)
  - 임대시세 (서울신용보증재단 데이터 기반)
- **프로젝트 활용 계획:**
  - 우리마을가게 상권분석서비스 데이터와 교차 검증
  - 더 세분화된 상권 단위 분석 가능
  - 사업자등록번호 기반 점포 데이터로 개별 매장 추적 가능성
- **주의사항:**
  - 데이터 업데이트 주기 확인 (분기별 2개월 후 업데이트)
  - 제공 형식 확인 (EXCEL, CSV, OPEN API, SHP)
  - 우리마을가게 데이터와의 중복 여부 확인
